Job Search Assistant

Where it is on the website

The Job Search Assistant is available at: https://ffjconsultingllc.com/job-search

What it does (simple, compliant approach)

This feature generates LinkedIn job search links based on either:

  • a short keyword phrase you type (example: aws architect)
  • or a longer resume text paste (to detect skills/titles)

Important

The app does not scrape LinkedIn and does not auto-apply for jobs. It creates clickable searches that open LinkedIn in a new tab.

How searches are generated

  1. Input: You type a preferred title or paste resume text.
  2. Detection: The app detects common skills (AWS, Terraform, Kubernetes, etc.) and roles.
  3. URL generation: It builds LinkedIn search URLs like: 
    https://www.linkedin.com/jobs/search/?keywords=Cloud+Architect+AWS+Terraform&location=United+States
  4. Open: You click a link (or use "Open top 3") to open LinkedIn in new tabs.

Filters (best-effort)

The UI includes toggles like Easy Apply and Remote only. These are encoded into the LinkedIn URL using common query parameters. LinkedIn can change these parameters over time. If a filter link ever stops working, simply re-run the search and apply filters in LinkedIn directly.

How to update/improve it

  • Frontend code: frontend/src/components/JobSearchAssistant.jsx
  • Styling: frontend/src/components/JobSearchAssistant.css

Future enhancements (optional)

  • Resume upload (PDF/DOCX) and text extraction
  • Better skill/title extraction using an LLM (Bedrock) behind a backend API
  • Job aggregation from sources that allow APIs (ATS feeds, public job boards)
