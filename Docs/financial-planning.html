Financial Planning (Portfolio Analysis)

Where it is on the website

The Financial Planning app is available at: https://ffjconsultingllc.com/financial-planning

What it does

This is a starter portfolio analysis tool designed to help you understand:

  • Portfolio allocation by holding
  • Market value, cost, and gain/loss
  • Top holdings concentration
Disclaimer: Educational purposes only. Not financial advice.

How to use it

  1. Add your positions (Ticker, Shares, Cost basis, Current price).
  2. Review computed totals (portfolio value, total cost, gain/loss).
  3. Use the allocation chart to see concentration.

Saved portfolios (multiple)

You can create multiple portfolios (e.g., Retirement, Trading, Long-term). These are stored in your browser (localStorage), so they persist across refreshes on the same device/browser.

Import / Export CSV

You can export your positions to CSV and re-import them later. Format: ticker,shares,costBasis,currentPrice

Delayed / manual data

This version is intentionally “delayed/manual friendly”: you can paste current prices from your broker or another source. Later we can add delayed quote lookups via a backend proxy if needed.
